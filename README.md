# piano-player
 Piano Player - это веб-приложение, реализующее виртуальное пианино.
Проект демонстрирует работу с Web Audio API, создание user-friendly интерфейса, а также запись и воспроизведение музыкальных последовательностей.

Пользователь может играть на пианино с помощью клавиатуры или мыши, записывать игру и сохранять её в формате JSON для последующего воспроизведения.
## Возможности сайта
### Interactive mode
 Интерактивный режим позволяет играть на пианино в реальном времени.
Возможности режима:
- игра с помощью клавиатуры
- игра с помощью мыши
- генерация звука через Web Audio API
- запись исполнения

Во время записи сохраняются:
- нажатые клавиши
- время нажатия
- длительность

Запись можно экспортировать в JSON, чтобы затем воспроизвести её.
### Prepared mode
Этот режим предназначен для воспроизведения заранее подготовленных музыкальных записей.
Возможности режима:
- воспроизведение JSON-композиций
- визуальное отображение нажатых клавиш
- управление воспроизведением:
- - запуск
- - пауза
- - остановка
- - изменение скорости

Записи могут:
- загружаться из JSON файла
- воспроизводиться из LocalStorage (если они были записаны ранее)

## Установка и запуск
```bash
https://github.com/bekulymba/piano.git
```
Запуск piano-player осуществляеется запуском локального сервера.  
### Вариант 1:  
Создаем python сервер
```css
python3 -m http.server 8000
```
Заходим в браузере:
```cs
http://localhost:8000
```

### Вариант 2:   
Расширение Live-server в VS code:  

Устанавливаем это [расширение](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)  
Заходим в папку проекта и справа снизу будет `Go Live`  
В браузере автоматически откроется сервер с проектом piano-player
