### Interactive mode
 Интерактивный режим позволяет играть на пианино вручную. Как клавишами клавишами клавиатуры, так и мышью.  
 Звук генерируется в реальном времени используя Web Audio API.
 В интерактивном режиме есть функция записи пианино, оно записывает нажатие клавиш и время что клавиша была нажата.  
 Записанный файл экспортируется в формате JSON, что позволит далее воспроизвести далее.
### Prepared mode
Заготовленный режим позволяет проиграть уже готовые песни в JSON Формате.
Присутствует визуальное сопровождение нажатия клавиш, а также 
Раннее записанная запись может быть воспроизведена в этом режиме, скачивание не обязательно, оно храниться в localstorage.
Есть элементы контроля записи: Запуск, пауза, стоп, скорость воспроизведения.
## Инструкция по запуску 
```bash
git clone git@git.platform.alem.school:bekulymba/piano-player
```
Запуск piano-player осуществляеется запуском локального сервера.  
### Вариант 1:  
Создаем python сервер
```css
python3 -m http.server 8000
```
Заходим в браузере:
```cs
http://localhost:8000
```

### Вариант 2:   
Расширение Live-server в VS code:  

Устанавливаем это [расширение](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)  
Заходим в папку проекта и справа снизу будет `Go Live`  
В браузере автоматически откроется сервер с проектом piano-player
